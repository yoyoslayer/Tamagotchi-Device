# Photophage. A pet that feeds on light

## What it is
I made a special digitel pet that you can carry around with you. It is like the Tamagotchi toys from the 90s but I built it myself and custom made. The pet is on a screen that comes with pressable buttons a buzzer, a LED, and a sensor that can tell if it is in the light or dark. All of this is inside an egg-shaped case that I made using a 3D printer. The case also takes inspiration and certain design elements from the AniManga, Soul Eater.
What makes my tamagotchi unique is that it can tell what is going on in your environment. It has a sensor that can tell if it present in sunlight, indoor lighting or if it is dark so it will react to where it is and not just to what you do with the buttons.

## What it does / how to use it
- There are three buttons that you can use to take care of the pet: Feed, Play, Sleep.
- The pet has three things that you need to keep track of: Hunger, Happiness, Energy. These will go down over time so you need to check on the pet
If you press Feed, the pets Hunger will go up.
If you press Play, the pets Happiness will go up. Its Energy will go down a little bit.
If you press Sleep the pets Energy will go up.


- The pets face will change depending on how it's feeling. It will be happy if it is doing well okay if it is getting by and sad if something is wrong.
- There is a light that will change color depending on how the pet is feeling: green means it is happy amber means it is okay and red means it needs help.
- When you press a button the pet will make a beep.
- The pet can tell if it is in sunlight or indoor light and it will show you this every minutes. If it is dark it will not say anything.
- The pet runs on a battery that you can recharge and it lives in a custom egg-shaped case that I made.

## Assembly
1. Download CAD files from onshape and the gerber zip.
2. Purchase the parts from the BOM or select alternatives. (Screws must be exact identicals in dimensions)
3. Order the PCB from JLCPCB (or your prefferred vendor) using the gerber file attached
4. Print the case enclosure that houses the PCB and components
5. Cut velcro tape to reasonable sizes and attach one half to the Battery enclosure in the bottom case and the other half to the battery.
6. Attach, assemble, and solder the parts to the PCB as it looks according to the Onshape CAD
   - Be sure to check your battery connection after soldering
   - Solder the battery cell into the bottom side of the PCB rather than the top
   - The JST PH connector's wire will need to be shortened significantly to fit in the bottom case or wrapped neatly. Solder the connector into the battery cell.
7. Insert heat inserts into their respective mounting holes, place PCB, then screw in the screws
8. Repeat step 6 for the case
9. Assembly complete. YAYYY

## Why I made it
I made Photophage because I wanted to try something challenging. This was my 1st time building a whole project from scratch and I wanted to add some special features like the light sensor and the mood light. I chose to make it a virtual pet that feeds on light because I think it is a fun and interesting idea. I used this project to learn about all the parts of building an electronics project from designing the circuit board to writing the code and making the case. This serves as my first submission to the program Fallout, created by HackClub, an organization that promotes and encourages teens to become engineers/programmers.
